# AI-Powered Lead Scoring and Sales Prediction CRM

## Overview

A Customer Relationship Management (CRM) system developed using Python and Django for managing leads, vehicles, quotations, and sales analytics. The project incorporates AI-based lead scoring, sales prediction, and recommendation features to help improve sales decision-making.

## Features

* Lead Management (Add, Edit, Delete, View Leads)
* Vehicle Management
* Follow-up Tracking
* Quotation Management
* AI-Based Lead Recommendations
* Lead Scoring System
* Sales Prediction Analytics
* Dashboard and Reports
* Variant-wise Lead Analysis
* User Authentication (Login/Register)

## Technologies Used

* Python
* Django
* SQLite
* HTML
* CSS
* Bootstrap
* JavaScript

## AI/ML Features

### Lead Scoring

Prioritizes leads based on customer details and engagement information.

### Sales Prediction

Predicts future sales trends using historical lead data.

### AI Recommendations

Provides suggestions such as:

* Call within 24 hours for high-priority leads
* Send quotation after test drive
* Offer referral benefits for referral leads

## Installation

1. Clone the repository

```bash
git clone https://github.com/Revathy-KV/Leadscore-and-sales-prediction.git
```

2. Navigate to the project directory

```bash
cd Leadscore-and-sales-prediction
```

3. Install dependencies

```bash
pip install django
```

4. Run migrations

```bash
python manage.py migrate
```

5. Start the server

```bash
python manage.py runserver
```

6. Open in browser

```text
http://127.0.0.1:8000/
```
